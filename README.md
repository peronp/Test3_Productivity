# Focus Timer

A simple Pomodoro-style focus timer built with HTML, CSS, and vanilla JavaScript.

## Features

- 25-minute focus sessions with short (5 min) and long (15 min) breaks
- Visual countdown ring
- Auto-switches to break mode when a session completes
- Audible beep on completion
- Configurable durations via the settings panel

## Usage

Open `index.html` in any browser — no build step or dependencies required.
